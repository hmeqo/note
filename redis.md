# Redis

| 命令 | 说明 |
| --- | --- |
| SET | 设置字符串值 |
| SETEX | 设置带有过期时间的字符串值 |
| SETNX | 设置字符串值, 如果已经存在没有过期则不做改变 |
| GET | 获取 |
| DEL | 删除 |
| KEYS | 获取全部或指定键 |
| TTL | 获取一个键目前的有效时间 |
| EXPIRE | 设置过期时间 |
| LPUSH | 将元素追加到列表头部 |
| RPUSH | 将元素追加到列表尾部 |
| LPOP | 弹出头部元素 |
| RPOP | 弹出尾部元素 |
| LLEN | 查看长度 |
| LRANGE | 切片 (包含头尾, 支持负数索引) |
| LTRIM | 裁剪 (切片, 去掉不包含的部分) |
