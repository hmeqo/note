# 数据类型

## 属性

| 类型                  | 描述       |
| --------------------- | ---------- |
| **数字类型**          |            |
| [`int`](#int)         | 整数类型   |
| [`float`](#float)     | 浮点数类型 |
| [`complex`](#complex) | 复数类型   |
| **可迭代类型**        |            |
| [`bool`](#bool)       | 布尔类型   |
| [`bytes`](#bytes)     | 字节类型   |
| [`str`](#str)         | 字符串类型 |
| [`list`](#list)       | 列表类型   |
| [`tuple`](#tuple)     | 元组类型   |
| [`dict`](#dict)       | 字典类型   |
| [`set`](#set)         | 集合类型   |

### int

整数类型

示例:

```python
a = 10
print(type(a))  # <class 'int'>
```

### float

浮点数类型

示例:

```python
a = 10.5
print(type(a))  # <class 'float'>
```

### complex

复数类型

示例:

```python
a = 1 + 2j
print(type(a))  # <class 'complex'>
```

### bool

布尔类型

示例:

```python
a = True
b = False
print(type(a))  # <class 'bool'>
```

### bytes

字节类型

示例:

```python
a = b'hello'
print(type(a))  # <class 'bytes'>
```

### str

字符串类型

示例:

```python
a = 'hello'
print(type(a))  # <class 'str'>
```

### list

列表类型

示例:

```python
a = [1, 2, 3]
print(type(a))  # <class 'list'>
```

### tuple

元组类型

示例:

```python
a = (1, 2, 3)
print(type(a))  # <class 'tuple'>
```

### dict

字典类型

示例:

```python
a = {'key': 'value'}
print(type(a))  # <class 'dict'>
```

### set

集合类型

示例:

```python
a = {1, 2, 3}
print(type(a))  # <class 'set'>
```
