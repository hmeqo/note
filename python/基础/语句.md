# 语句

## 属性

| 语句类型                      | 描述           |
| ----------------------------- | -------------- |
| **定义函数和类**              |                |
| [`def`](#def)                 | 定义函数       |
| [`class`](#class)             | 定义类         |
| [`decorator`](#decorator)    | 装饰器         |
| [`lambda`](#lambda)           | 匿名函数       |
| [`global`](#global)           | 全局变量       |
| [`nonlocal`](#nonlocal)       | 上层变量       |
| [`return`](#return)           | 返回值         |
| [`yield`](#yield)             | 生成器         |
| **异常处理**                  |                |
| [`try`](#try)                 | 捕获异常       |
| [`assert`](#assert)           | 断言           |
| [`raise`](#raise)             | 抛出异常       |
| **导入模块**                  |                |
| [`import`](#import)           | 导入模块       |
| [`from_import`](#from_import) | 导入模块或属性 |
| **循环语句**                  |                |
| [`for`](#for)                 | 循环语句       |
| [`while`](#while)             | 循环语句       |
| [`break`](#break)             | 退出循环       |
| [`continue`](#continue)       | 退出本轮循环   |
| **条件判断**                  |                |
| [`if`](#if)                   | 条件判断语句   |
| **其他**                      |                |
| [`as`](#as)                   | 别名           |
| [`del`](#del)                 | 删除对象       |
| [`pass`](#pass)               | 占位符         |
| [`with`](#with)               | 上下文管理器   |

### def

定义函数

示例:

```python
def func_name(*args, **kwargs):
    pass
```

### class

定义类

示例:

```python
class ClassName:
    pass
```

### decorator

装饰器

示例:

```python
def decorator(func):
    return func

@decorator
def func():
    pass
```

### lambda

匿名函数

示例:

```python
lambda x: x + 1
```

### global

全局变量

示例:

```python
global var
```

### nonlocal

上层变量

示例:

```python
nonlocal var
```

### return

返回值

示例:

```python
def func():
    return 1
```

### yield

生成器

示例:

```python
def generator():
    yield 1
```

### try

捕获异常

示例:

```python
try:
    pass
except:
    pass
```

### assert

断言

示例:

```python
assert condition
```

### raise

抛出异常

示例:

```python
raise Exception()
```

### import

导入模块

示例:

```python
import module
```

### from_import

导入模块或属性

示例:

```python
from module import func
```

### for

循环语句

示例:

```python
for i in range(10):
    pass
```

### while

循环语句

示例:

```python
while condition:
    pass
```

### break

退出循环

示例:

```python
for i in range(10):
    if i == 5:
        break
```

### continue

退出本轮循环

示例:

```python
for i in range(10):
    if i == 5:
        continue
```

### if

条件判断语句

示例:

```python
if condition:
    pass
```

### as

别名

示例:

```python
import module as m
```

### del

删除对象

示例:

```python
del var
```

### pass

占位符

示例:

```python
pass
```

### with

上下文管理器

示例:

```python
with open('file') as f:
    pass
```
