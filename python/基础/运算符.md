# 运算符

## 属性

| 运算符                 | 描述     |
| ---------------------- | -------- |
| **算术运算符**         |          |
| [`+`](#plus)           | 加法     |
| [`-`](#minus)          | 减法     |
| [`*`](#multiply)       | 乘法     |
| [`/`](#divide)         | 除法     |
| [`//`](#floor-divide)  | 整除     |
| [`%`](#modulo)         | 取余     |
| [`**`](#power)         | 幂运算   |
| **赋值运算符**         |          |
| [`=`](#assign)         | 赋值     |
| **比较运算符**         |          |
| [`==`](#equal)         | 等于     |
| [`!=`](#not-equal)     | 不等于   |
| [`<`](#less)           | 小于     |
| [`>`](#greater)        | 大于     |
| [`<=`](#less-equal)    | 小于等于 |
| [`>=`](#greater-equal) | 大于等于 |
| **逻辑运算符**         |          |
| [`and`](#and)          | 与       |
| [`or`](#or)            | 或       |
| [`not`](#not)          | 非       |
| **位运算符**           |          |
| [`&`](#bitwise-and)    | 按位与   |
| [`\|`](#bitwise-or)    | 按位或   |
| [`^`](#bitwise-xor)    | 按位异或 |
| [`~`](#bitwise-not)    | 按位取反 |
| [`<<`](#left-shift)    | 左移     |
| [`>>`](#right-shift)   | 右移     |

### + {#plus}

示例:

```python
a = 1 + 2
print(a)  # 3
```

### - {#minus}

示例:

```python
a = 2 - 1
print(a)  # 1
```

### * {#multiply}

示例:

```python
a = 2 * 3
print(a)  # 6
```

### / {#divide}

示例:

```python
a = 6 / 2
print(a)  # 3.0
```

### // {#floor-divide}

示例:

```python
a = 7 // 2
print(a)  # 3
```

### % {#modulo}

示例:

```python
a = 7 % 2
print(a)  # 1
```

### ** {#power}

示例:

```python
a = 2 ** 3
print(a)  # 8
```

### = {#assign}

示例:

```python
a = 1
print(a)  # 1
```

### == {#equal}

示例:

```python
a = 1 == 1
print(a)  # True
```

### != {#not-equal}

示例:

```python
a = 1 != 2
print(a)  # True
```

### < {#less}

示例:

```python
a = 1 < 2
print(a)  # True
```

### > {#greater}

示例:

```python
a = 2 > 1
print(a)  # True
```

### <= {#less-equal}

示例:

```python
a = 1 <= 1
print(a)  # True
```

### >= {#greater-equal}

示例:

```python
a = 2 >= 1
print(a)  # True
```

### and {#and}

示例:

```python
a = True and False
print(a)  # False
```

### or {#or}

示例:

```python
a = True or False
print(a)  # True
```

### not {#not}

示例:

```python
a = not True
print(a)  # False
```

### & {#bitwise-and}

示例:

```python
a = 1 & 0
print(a)  # 0
```

### | {#bitwise-or}

示例:

```python
a = 1 | 0
print(a)  # 1
```

### ^ {#bitwise-xor}

示例:

```python
a = 1 ^ 0
print(a)  # 1
```

### ~ {#bitwise-not}

示例:

```python
a = ~1
print(a)  # -2
```

### << {#left-shift}

示例:

```python
a = 1 << 1
print(a)  # 2
```

### >> {#right-shift}

示例:

```python
a = 2 >> 1
print(a)  # 1
```
