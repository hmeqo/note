# 编码规范

* 一行只写一条语句
* 用 {} 括起来的部分表示程序的某一层次结构。"{" 一般放在这一结构开始行的结尾处, "}" 与该结构的每个字母对齐, 并独占一行
* 低一层次的语句或注释应该比高一层次的语句或注释缩进若干个空格后再书写, 使程序更加清晰, 增加程序源代码的可读性
